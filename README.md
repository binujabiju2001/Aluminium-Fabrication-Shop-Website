

GS Aluminium Fabrication Website

Description
The GS Aluminium Fabrication Website is a modern and user-friendly web application designed to showcase the services offered by GS Aluminium Fabrication. The website provides a seamless experience for users to explore services, view contact details, and learn more about the shop’s offerings.

Features
- Home Page: Overview of the shop and its services.
- Services Page: Detailed information about the various fabrication services provided.
- Contact Page: Contact information and a simple form for inquiries.
- Responsive Design: Ensures the website works smoothly on all devices.

  - CSS3
